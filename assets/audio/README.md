# Diva Academy - Music Tracks

Place your music files in this directory with the following names:

## Required Tracks

| Filename | Situation | Suggested Source |
|----------|-----------|------------------|
| `title.wav` | Title screen (loopable) | Pixabay: "After Party" or "Ultralounge" |
| `lobby.mp3` | Lobby/exploration (loopable) | Pixabay: "Mysterious Mystical Ambient" |
| `battle.mp3` | Battle (loopable) | Pixabay: "Synth Pop" |
| `victory.mp3` | Victory stinger (~5s) | Freesound: "Orchestral Victory Fanfare" (CC0) |
| `gameover.wav` | Game over (~5s) | Freesound: search "dramatic sting" CC0 |

## Notes

- Loopable tracks will be looped automatically by the game
- Short tracks (victory, gameover) play once and fade out
- If tracks are not found, the game falls back to procedural Web Audio music
- Both WAV and MP3 formats are supported
